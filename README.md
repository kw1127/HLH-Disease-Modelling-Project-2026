# HLH Disease Modelling Project 2026

<big> This repository contains the code for the exploratory data analysis (EDA) of Haemophagocytic Lymphohistiocytosis (HLH), a life-threatening hyperinflammatory syndrome. In its primary form, HLH is caused by genetic mutations that impair the perforin-granzyme cytotoxic pathway in natural killer (NK) cells and cytotoxic T lymphocytes (CTLs). This prevents effective apoptosis of target cells, resulting in a failure to resolve the immune response. The persistent immune activation drives uncontrolled proliferation of T cells and macrophages, which secrete excessive levels of proinflammatory cytokines including IFN-γ, IL-1, IL-6, IL-18, and TNF-α, which result in a cytokine storm. The hyperactivated macrophages/histiocytes phagocytose blood cells (hemophagocytosis), leading to clinical features such as cytopenias, hepatosplenomegaly, hyperferritinaemia, and persistent fever, which can progress to multi-organ failure if untreated.

This project will work with both clinical and single-cell RNA sequencing (scRNA-seq) data for HLH to gain insights into rare and/or interesting clusters which could serve as basis for novel therapeutic development.
